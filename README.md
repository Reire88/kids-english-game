KidsEnglishGame (local)
======================

A simple kids English speaking game (age ~5), 10 questions per round.
- Tap mic, say the word in English.
- 1 retry if wrong.
- Forgiving matching (articles, plural, soundex + levenshtein).
- Success/fail sounds.
- Prize vehicle based on score.

How to run
----------
1) Install Android Studio
2) Open this folder as a project
3) Let Gradle sync
4) Run on your Samsung phone (USB debugging) or emulator

Tip: On a real device, SpeechRecognizer works best with Google app installed/enabled.
